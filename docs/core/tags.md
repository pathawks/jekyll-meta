Similar to categories, one or multiple tags can be added to a post. Also like categories, tags can be specified as a [YAML list](http://en.wikipedia.org/wiki/YAML#Lists) or a space- separated string.

{% highlight yaml %}
---
tags: [docs, core]
---
{% endhighlight %}
